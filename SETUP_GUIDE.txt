USEFUL COMMANDS


Database:
  php artisan migrate              # Run migrations
  php artisan db:seed              # Run all seeders
  php artisan db:seed --class=CategorySeeder  # Run specific seeder
  php artisan migrate:fresh --seed # Reset and seed

Admin:
  php artisan make:filament-resource ModelName --generate

Models:
  php artisan make:model ModelName -m
  php artisan make:controller ControllerName

Maintenance:
  php artisan cache:clear
  php artisan config:clear
  php artisan view:clear

TROUBLESHOOTING


Problem: Port 8000 already in use
Solution: php artisan serve --port=9000

Problem: Admin panel not loading
Solution: php artisan cache:clear && php artisan config:clear

Problem: Database errors
Solution: php artisan migrate:fresh --seed

Problem: Images not showing
Solution: php artisan storage:link
