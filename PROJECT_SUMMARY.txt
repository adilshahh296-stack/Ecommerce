 QUICK COMMANDS

  Database:
    php artisan migrate              # Run migrations
    php artisan db:seed              # Seed sample data
    php artisan migrate:fresh --seed # Reset everything

  Server:
    php artisan serve                # Start server
    php artisan tinker               # PHP shell

  Maintenance:
    php artisan cache:clear          # Clear cache
    php artisan config:clear         # Clear config
    php artisan view:clear           # Clear views

  Model Generation:
    php artisan make:model MyModel -m         # Model + migration
    php artisan make:controller MyController   # Controller
    php artisan make:filament-resource MyModel --generate  # Admin



 CUSTOMIZATION EXAMPLES

  1. Add New Product Field:
     - Create migration: php artisan make:migration add_field_to_products
     - Run migration: php artisan migrate
     - Update Product model fillable array
     - Update Filament resource

  2. Change Colors:
     - Edit Tailwind classes in blade files
     - Edit colors in resources/views/app.blade.php

  3. Add New Category:
     - Go to admin panel  Categories  Create
     - Appears instantly on frontend

  4. Add Payment Gateway:
     - Install Stripe/PayPal package
     - Add checkout controller
     - Update routes
     - Create checkout view
